# :bookmark: Library

**Live Demo:** https://jlionperez.github.io/odin-library/

A dynamic book tracking application built with JavaScript, HTML, and CSS.

## :pencil: Features

- Add books with title, author, and page count
- Toggle read status on individual book entries
- Remove books from the library
- Unique ID generation for each entry
- Book entry form utilizing a modal with real-time UI updates

## :computer: Tech Stack

- JavaScript (ES6)
- HTML5
- CSS3

## :book: Concepts Applied

- Object-Oriented Programming using ES6 classes with constructor, getters, and setters
- cryto.randomUUID() for unique ID generation
- Modal dialog with form validation and event prevention
- Dynamic table rendering with interactive controls

## :gear: How to Run Locally

1. Clone the repository:   
```git clone https://github.com/JLionPerez/odin-library.git```

2. Navigate into the project folder:   
```cd odin-library```

3. Open the ```index.html``` file in your browser.
